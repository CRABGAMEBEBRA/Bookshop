ЗАДАНИЕ

В рамках проекта необходимо:

Сверстать главную страницу интернет-магазина Bookshop.
Подключить Google Books API так, чтобы данные о книгах подгружались с сервера.
Подключить слайдер.
Поработать над правильной организацией проекта: . реализовать модульную структуру; . подключить Webpack; . настроить сборку с минификацией.
Применить различные инструменты оптимизации.


Create webpack

1) npm init -y
2) npm install webpack webpack-cli webpack-dev-server --save-dev
3) npm install style-loader css-loader --save-dev
4) npm install html-webpack-plugin --save-dev

Как запускать проект:
1)npm install
2)npm run build
3)запускать проект через index.html
